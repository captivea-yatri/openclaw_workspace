Ocean.io Connector

Overview

Integrate Ocean.io with Odoo to fetch API tokens, build secure webhook endpoints, and enrich contacts, leads and companies automatically. Manage multiple Ocean.io instances, monitor credits, and run async reveals for emails and phones.

How It Works

01

Configure an Ocean.io Instance in Odoo: enter your API token, set the public HTTPS URL (or let Odoo detect it), and the module automatically generates a secure webhook URL and secret.


02

Use the provided wizards (People Lookup, Lead Prospect, Company Enrichment) to search Ocean.io by LinkedIn, email, domain or name. Results are displayed instantly and can be applied to Odoo contacts or leads.


03

Enable async reveals for verified email or phone numbers. Ocean.io posts the results back to the secure webhook; the module logs the payload and updates the linked Odoo record automatically.
