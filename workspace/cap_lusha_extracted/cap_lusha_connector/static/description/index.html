Lusha Connector

Overview

Connect Odoo with Lusha API V3 to enrich contacts, enrich companies, prospect leads, retrieve signals, and receive webhook notifications – all from within Odoo.

How It Works

01

Create a **Lusha Instance** in Odoo – enter your API key, optional public URL, and the module will generate a secure webhook endpoint and secret.


02

Use the Contact Enrichment wizard to search Lusha by email, LinkedIn URL, or name + company. The wizard returns emails, phone numbers, job titles and other details, which can be applied directly to the selected Odoo contact.


03

Use the Company Enrichment wizard to look up a company by name or domain. Lusha returns industry, employee count, revenue range, headquarters location and more, which can be written back to the Odoo partner record.


04

Use the Lead Prospecting wizard to run advanced filters and retrieve a list of potential leads. The results can be turned directly into Odoo CRM leads with full contact details.


05

Enable **async reveals** – Lusha can push verified email or phone numbers back to Odoo via the webhook. The payload is logged and the linked record is updated automatically.


Features Supported through this Module:

1) Lusha Instance Management (API key, webhook URL, credit balance)

2) Contact Enrichment (email, LinkedIn, phone, job title)

3) Company Enrichment (industry, size, revenue, domain)

4) Lead Prospecting & Lookalikes

5) Signals (job changes, hiring, news)

6) Async Email / Phone Reveal via Secure Webhook

7) Credit & Rate‑Limit Monitoring

8) Detailed Webhook Logging for Auditing
